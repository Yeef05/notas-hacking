# MatchTheRegex

## Descripción

How about trying to match a regular expressionThe website is running [here](http://saturn.picoctf.net:59208/).
## Solución

```
Inspeccionamos en el código y vemos que busca una expresión regular con el siguiente critero
^p.....F!?

Usamos una pagina para buscar una expresión regular valida y nos da esta
p12345F

Obtenemos la flag de la página web
picoCTF{succ3ssfully_matchtheregex_f89ea585}
```
